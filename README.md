# MurVeg
Android mobile app - Murmansk vegetation catalog

Мобильное приложениее позволяет просматривать каталог растительности Мурманска, просматривая описание и фото, искать растительность по ключевым словам, добавлять элементы каталога в  "избранное".
Приложение работает оффлайн, данные сохраняются локально.
Поддержка Android 7.0 (API 24) и выше.
Интерфейс на русском языке.
Технические характеристики:
- Язык: Kotlin.
- UI: Jetpack Compose. 
- Архитектура: MVVM с использованием ViewModel и Repository
- Локальное хранение: Room Database
